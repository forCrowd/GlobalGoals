An interactive crowdfunding platform for projects/ideas about Sustainable Developer Goals (SDGs).

By using this platform, the users can vote for the importance of three categories:

  • SDG: Which of the SDGs should be focused and solved first?
  • Legal Entity: Business model of the project can be "non-profit" / "for-profit". Which type of legal entity should get more funding?
  • License: If the project is about developing a software, which type of license should get more funding?

Additionally, users can also choose the importance of each category.

Based on these results, the platform distributes the funds to all ideas.

Global Goals Fund is a custom prototype that we developed and presented at United Nation's Connect2Effect Hackathon.
